# edu-docker-env

## Pull

```
docker pull rcmlz/edu-docker-env-minimal
```

```
docker pull rcmlz/edu-docker-env-full
```

## Get Source
```
git clone --depth 1 https://github.com/rcmlz/edu-docker-env
cd edu-docker-env
```

## Start

````
docker compose -f docker-compose-full.yml up
````

````
docker compose -f docker-compose-minimal.yml up
````

## Build and Launch Yourself

````
./build-and-launch-minimal.sh
./build-and-launch-full.sh
````

````
./launch-minimal.sh
./launch-full.sh
````
